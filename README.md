# Newsubs
A tool to find newly deployed subdomains

## Why?
Every company create or update and deploy new subdomains on the web. To sort out newly deployed subdomains, `newsubs` tools helps to find it out. 

## Usage
```
Usage: newsubs [-f] <file> [-s] <file> [-o] <filename>
-h: Help
-f --first: First File
-s --second: Second File
-o --output: Output file name/path
```

## Install
```
git clone https://github.com/heydc7/newsubs.git
cd newsubs/
python3 main.py -f first.txt -s second.txt
```

## Example
```
python3 main.py -f first.txt -s second.txt -o output.txt
```
